# Olist Orders Dashboard

This project analyzes the Olist e-commerce orders dataset. It visualizes delivery times, top-selling products, seller revenue, and monthly revenue trends using Python.

---

## Features

- Distribution of delivery times
- Top 10 most sold products
- Top 10 sellers by revenue
- Monthly revenue trends
- Uses Python libraries: pandas, numpy, matplotlib, scikit-learn

---

## Requirements

- Python 3.x  
- pandas  
- numpy  
- matplotlib  
- scikit-learn  

Install dependencies with:

```bash
pip install pandas numpy matplotlib scikit-learn
